About this Site.

This site was built on top of a Bootstrap framework. This framework gives the site some basic responsiveness to different screen sizes and offers a basic CSS and JavaScript toolkit to draw from when building the site. I chose Bootstrap because it is quick to implement, I have used it many times, and I know it works well across browsers and devices.

The custom stylesheet for this site was built using SASS. SASS allows for CSS customization using inheritance, variable declaration, and a fair amount of encapsulation. This not only saves time on me as a developer, it saves on technical debt as the code grows. With SASS (, PHPStorm, and Node.js) I can even minify my CSS files as I debug my stylesheet.

Finally, this site was held together by a not-so-complex sequence of pushes and pulls (and commits). GitHub kept my work backed up and allowed me to transition this site from my laptop to Amazon Web Services. Plus, GitHub allows my work to be open source so that maybe other developers can learn from this site the same way I have learned from so many of the web sites that came before this one.
